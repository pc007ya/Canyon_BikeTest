步驟 07Step 07

對應設備怎麼查How to look up equipment

導覽列 「設備盤點」 頁列出每項設備用在哪些測試。需求量為所有測試的 最高單次用量（設備共用、不累加）。The Equipment page lists which tests use each machine. Demand is the peak per-test usage across all tests (shared, not summed).

致動器用於 13 項測試，但需求 = 最高的 2 支，不是相加 26。The actuator is in 13 tests, but demand = the peak 2, not 26 summed.

重點回顧Recap

十步走完，DEMO 帳號全流程就緒Ten steps — the DEMO flow, end to end

登入 → 目前專案：選 DEMO、輸入密碼，只看被分配的案子。Login → Projects: pick DEMO, enter the code, see only assigned work.
制具＋設備 → 安裝：核對清單，上傳實拍對比官方圖。Fixtures + equip → setup: check the lists, upload a photo beside the official diagram.
報告 → 查詢 → 盤點：填結果送出；制具累加、設備取最高。Report → lookup → stock-take: submit; fixtures sum, equipment takes the max.
變更通知：留意鈴鐺紅點，粉紅＝短缺或項目被變更需重新確認。Alerts: watch the bell; pink = shortfall or a changed item to re-check.
